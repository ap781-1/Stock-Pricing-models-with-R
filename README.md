## Welcome to Pricing models with R repository.
In this repository you will find various models with enhancements.

Advanced Monte Carlo Framework for Multi-Model Option Pricing : This is a comprehensive Monte Carlo option pricing framework in R that combines multiple quantitative finance models, numerical techniques, and performance enhancements into one unified tool.

The script builds an advanced option pricing engine that can:

Price European and American options, Use multiple stochastic models for asset price, like Geometric Brownian Motion (GBM),Heston (stochastic volatility), Merton (jump diffusion)

Improve accuracy using Antithetic variates (variance reduction), Control variates (uses Black-Scholes as a benchmark)

Provide: Confidence intervals, Error estimates, Convergence diagnostics and Visualization

Other code builds a flexible and base models for pricing system that Prices complex derivatives.
